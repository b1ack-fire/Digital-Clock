Digital Clock

This is a simple and clean digital clock application built using HTML, CSS, and JavaScript.
It displays the current time, date, and day, and also includes a basic alarm system, a countdown timer, and a theme toggle with a smooth animated switch.

Features
Real-Time Digital Clock

The clock updates every second and shows:

Hours

Minutes

Seconds

AM/PM

Current date

Day of the week

Alarm

Allows the user to set a custom alarm time

Plays a ringtone when the alarm triggers

Includes support for custom online audio sources (such as YouTube-to-MP3 CDN links)

Timer

Users can enter a number of seconds

A countdown will begin immediately

An alert sound will play when the timer finishes

Theme Toggle

Light and dark themes

Smooth animated slide-switch

The switch changes appearance (sun and moon icons) depending on the mode
